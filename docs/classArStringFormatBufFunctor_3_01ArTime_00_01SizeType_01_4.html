template<typename SizeType>
class ArStringFormatBufFunctor< ArTime, SizeType >

Specialization of ArStringFormatBufFunctor for ArTime.

The format string should match two unsigned long value, seconds and additional milliseconds. (e.g. "%lu.%lu")
