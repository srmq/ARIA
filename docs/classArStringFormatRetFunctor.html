Detailed Description

template<typename T>
class ArStringFormatRetFunctor< T >

ArRetFunctor subclass which returns a formatted std::string using value obtained by calling an accessor (via supplied ArRetFunctor) The value is formatted using sprintf, using the format string provided.

The resulting string must be less than 1024 bytes.
