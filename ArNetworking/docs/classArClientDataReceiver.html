Detailed Description

template<class C, typename T>
class ArClientDataReceiver< C, T >

A client program may use this to receive responses to data requests and pass the received data to a provided function or method, or save (cache) the most recently received data.

See also
ArServerDataProvider
Note
If the server does not have the requested data item at time of request, it will not be requested. A warning will be logged.

2.9.3
